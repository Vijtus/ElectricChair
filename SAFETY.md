# Safety

This project controls moving hardware.

Use it only when you can stop the chair by physical means.

Minimum checks:

- keep mains power easy to disconnect
- test one command at a time
- keep hands, tools, and cables away from moving parts
- confirm serial voltage levels before wiring
- do not trust the web UI as a safety device
- do not expose the server to untrusted networks
- treat `notes/` as lab notes, not as a final specification

Start simple. Observe the hardware. Change one thing at a time.
